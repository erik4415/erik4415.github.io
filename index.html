Personal Projects

Find a Good Neighborhood Website

This is a website I built using the google maps api to find a good place to live in any city. I moved a couple of years ago to a new city and had no idea what were safe areas and good neighborhoods were. I ended up looking at crime statistics for the city which helped oreint me to the city. Unfortunately crime statistic API's differ from state to state and I wanted something that would work anywhere. This map uses searches of the surrounding area to find business and institutions that correlate with safer areas. As an example if there is a Whole Foods in your neighborhood, it probably has less crime. This really more indicates socioeconomic status of the areas, but this was a fun project to play around with and work with googles API's. After entering a city to search on the map areas in green are more likely to have less crime, and red areas will potentially have more.

Audiogram Website

My college degree was in Speech/Language/Hearing sciences, which pretty much leads to either grad school for Speech Pathology or Audiology (Ear Doctor). I started grad school for Audiology but quickly realized that it wasn't for me. One of the main assessments in Audiology is using an audiogram to record the hearing levels of a patient. I created this simple to use online version mainly to practice with javascript, and it allows for a lot easier reading of an audiogram and quick documentation. Many of the students in the UW Hearing Sciences division started using this during clinical hours to document audiograms.

Noteworthy Academic Projects

Word Cloud Generator - Created this in my data structures class. This program takes a text file and finds the most common words in that file, then creates an html file that makes the words bigger based on how many times they appear in the text. Since this was a class project we were required to use certain data structures, so it is not the most efficient design and uses a combination of priority queues and AST's. It was an interesting project though and I loved that the end result was something that could be useful.

Custom Programming Language Compiler - Made this as a semester long project for my compiler and programming class. The syntax is a combination of C and java, and uses the Java VM to compile too. Main concepts I learned from this indcluded regular expressions, CUP specifications, finite automata, and actual code generation. Also learned about code optimization at the compiler level and general low level programming concepts.

3D Tic Tac Toe - What could make tic tac toe better? Playing it in 3D obviously. This is a text based tic tac toe game that uses three boards instead of one. So getting 3 in a row can now involve the extra dimension of backwards and forwards.
